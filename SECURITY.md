# Security policy

## Supported versions

| Version | Supported |
| --- | --- |
| latest `@derived-modular/*` on npm | yes |
| older majors | best effort |

## Reporting a vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report privately by opening a [GitHub Security Advisory](https://github.com/mikhailmogilnikov/derived-modular-architecture/security/advisories/new) or emailing the repository owner via GitHub profile contact.

Include:

- Affected package (`@derived-modular/cli`, `eslint-plugin`, etc.)
- Steps to reproduce
- Impact assessment if known

We aim to acknowledge reports within a few business days.
