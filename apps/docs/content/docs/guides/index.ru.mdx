## По фреймворкам

Каждый гайд — тот же mini-shop, разный корень композиции:

| Гайд | Корень композиции | Пример в репо |
| --- | --- | --- |
| [Vite + React](/guides/vite-react) | `src/app/` | Запускаемый canon |
| [Next.js](/guides/nextjs) | `src/app/` | App Router + RSC |
| [Astro](/guides/astro) | `src/pages/` + `src/app/` | `.astro` frontmatter |
| [SvelteKit](/guides/sveltekit) | `src/routes/` | Svelte 5 runes |
| [Vue + Vite](/guides/vue) | `src/app/` | SFC + composables |

Все примеры проходят `dma check` — исходники в [examples](https://github.com/mikhailmogilnikov/derived-modular-architecture/tree/main/examples).

## Общее для всех стеков

- Роуты тонкие — импорт только из `*/public/*`
- Нет `feature → feature`
- `services/` создаётся при promotion, не заранее
- CI: `dma check` + линтер по выбору
