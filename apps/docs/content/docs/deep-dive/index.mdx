Deep Dive is for people who already understand the basics and want answers to "why it works this way." You do not have to read it cover to cover. Pick the question that came up in review or in a debate with a teammate.

<Callout type="info">
  If DMA is new to you, start with [What is DMA](/start/what-is-dma) and [Four invariants](/concepts/invariants). You can come back here later.
</Callout>

## Recommended order

If you do read it in sequence:

1. [Why two module layers](/deep-dive/why-two-layers) — layer philosophy
2. [Why no barrel files](/deep-dive/why-no-barrels) — graph transparency
3. [Services vs Shared](/deep-dive/services-vs-shared) — the only "gray" boundary
4. [Why CLI, not only linters](/deep-dive/why-cli-not-only-linters) — enforcement
5. [DMA vs FSD](/deep-dive/dma-vs-fsd) — comparison with an alternative
