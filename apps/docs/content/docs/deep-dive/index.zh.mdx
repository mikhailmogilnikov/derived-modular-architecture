3. [Services vs Shared](/deep-dive/services-vs-shared) — 唯一「灰色」边界
4. [为什么需要 CLI，而不只靠 linter](/deep-dive/why-cli-not-only-linters) — 强制校验
5. [DMA vs FSD](/deep-dive/dma-vs-fsd) — 与另一种方案的对比
