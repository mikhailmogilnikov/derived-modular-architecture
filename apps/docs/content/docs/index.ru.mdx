Фронтенд-архитектура, где **правила выводятся из файловой системы и графа импортов**, а затем проверяются инструментами — а не вкусом.

```text
src/
├── app/         # composition root
├── features/    # leaf modules
├── services/    # modules with inbound edges
└── shared/      # portable ui / lib / api / model
```

## Четыре инварианта

1. Импорты только вниз: `app → features → services → shared`
2. Публичный API без barrel-файлов — межмодульные импорты идут в `*/public/*`
3. Колокация по умолчанию
4. Правило второго использования — выносить только при появлении второго потребителя
